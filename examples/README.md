# Examples

Representative example payloads for development and testing.

## Files

| File | Description |
|------|-------------|
| `candidate-profile.json` | Example user/candidate profile |
| `conversation-state.json` | Mock conversation state machine |
| `feedback-report.json` | Example AI feedback report |
| `interview-session.json` | Example interview session |
| `llm-response.json` | Example LLM API response |

## Rules

- Examples are realistic but contain no real PII
- Keep examples as small as needed to be useful
- Used in tests, documentation, and API exploration
